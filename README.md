musicPlayer
===========
你需要修改config.ini中的mp3路径

Dependencies
============
需要configparser 库

eassy_install configparse

本代码在python3.3 +qt5.1上开发，均为64位

图片都为图片格式，未转二进制，方便大家好看哈，各位看官可自行改着玩就好了
